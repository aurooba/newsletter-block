A native newsletter block that currently integrates with Convertkit.

== Description ==

Provides a single block that integrates with Convertkit for a newsletter opt-in form you can add in the Block Editor.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/newsletter-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Grab your [API Key](https://app.convertkit.com/account/edit) from Convertkit and add it to the Convertkit API Key field in the _General_ Settings.
4. Open up the Block Editor and search for the _Newsletter_ block.
